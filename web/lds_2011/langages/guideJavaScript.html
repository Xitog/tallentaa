Introduction

C’est un langage de programmation, dont le nom officiel est ECMAScript. Sa syntaxe s’inspire de Java et C++ mais ces concepts sont totalement différent. Le standard officiel est ECMA-262, adopé par l’ECMA en Juin 1997, reconnu par l’ISO en 1998 sous le nom de ISO/IEC 16262.

3. Instructions

Elle se termine par un point-virgule. ex: document.write(“Hello”); Le code est une suite d’instructions.

3.1. Bloc

Un bloc est un ensemble d’instructions entourées d’une paire d’accolade.

{
    document.write(“Hello”);
    document.write(“World”);
}
