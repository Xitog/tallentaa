Le code

La sélection

      if condition then actions A [ else actions B ] end

Exécute si la est vraie. Sinon, exécute (si spécifiée).

      unless condition then actions [ else actions B ] end

On traduire unless par "à moins que" ou "sauf si". Exécute si la est fausse. Sinon, exécute (si spécifiée).

Remarque : unless c = if not c
